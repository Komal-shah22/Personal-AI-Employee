# Company Handbook

## Email Communication Rules

### Tone
- Professional but friendly
- Clear and concise
- No jargon unless necessary
- Always proofread before sending

### Signature Format
```
Best regards,
[Your Name]
[Your Title]
[Company Name]
[Contact Information]
```

## WhatsApp Response Style

- Quick acknowledgment within 1 hour during business hours
- Use emojis sparingly and professionally
- Keep messages brief and actionable
- For complex topics, suggest email or call
- Business hours: 9AM - 6PM PKT

## Payment Approval Rules

### Thresholds
- **Under PKR 10,000**: Auto-approve if within budget category
- **PKR 10,000 and above**: Requires explicit approval
- **Recurring payments**: Review quarterly

### Required Information
- Invoice/receipt
- Vendor name
- Purpose/category
- Budget line item

## Social Media Posting Rules

### Content Guidelines
- No political content
- No controversial topics
- Maximum 5 hashtags per post
- Include relevant images/media when possible
- Tag relevant accounts appropriately

### Posting Schedule
- **Allowed hours**: 9AM - 6PM PKT
- **Frequency**: Maximum 3 posts per day
- **Best times**: 10AM, 2PM, 5PM

### Platform-Specific Rules
- **LinkedIn**: Professional content only
- **Twitter**: Industry news, insights, updates
- **Instagram**: Visual content, behind-the-scenes

## Security Rules

### Critical Rules
- ❌ NEVER store passwords in vault files
- ❌ NEVER commit API keys to git
- ❌ NEVER share sensitive data via unsecured channels
- ✅ Use environment variables for secrets
- ✅ Encrypt sensitive documents
- ✅ Use 2FA wherever possible

### Data Handling
- Personal data: Handle according to privacy policy
- Financial data: Encrypted storage only
- Client data: Confidential, need-to-know basis

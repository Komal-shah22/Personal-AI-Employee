# Skill: update-dashboard

**Name**: update-dashboard

**Description**: Refreshes Dashboard.md with real-time statistics and activity

**Version**: 1.0

**Trigger Phrases**:
- "update dashboard"
- "refresh dashboard"
- "show status"
- "system status"
- "refresh stats"

## Purpose
This skill refreshes the Dashboard.md file with real-time statistics about the AI Employee system, including task counts, priorities, recent activity, and system health metrics.

## Behavior
1. Counts tasks in different states:
   - Pending tasks: Count .md files in /Needs_Action
   - Planned tasks: Count .md files in /Plans
   - Completed today: Count .md files in /Done modified today
   - Total processed: Sum of all tasks ever handled

2. Identifies high-priority items:
   - Scan all files in /Needs_Action
   - Filter for priority: high
   - Collect first 5 for display

3. Gets recent activity:
   - Read /Logs/[today].txt
   - Extract last 10 entries
   - Format as bulleted list

4. Calculates statistics:
   - Average completion time (if data available)
   - Most common task types
   - Peak activity hours

5. Writes formatted Dashboard.md file

- If folders don't exist, create them
- If no data available, show zeros instead of errors
- If log files missing, show "No activity logged yet"
- Always succeed even with partial data

## Rules
- Update timestamp to exact current time
- Keep formatting consistent
- Don't include sensitive information in dashboard
- Sort tasks by priority then by time
- Limit lists to reasonable lengths (max 10 items)
